JavaScript:

$('.tabbar').tabScrollify();

Documentation

Syntax:

$(element).tabScrollify(options);

Options:

  • tabSelector - (string, defaults to 'a') A string indicating the elements that resembles the tab, which must have widths.
  • prevText - (string, defaults to '&laquo;') A string indicating the HTML for the 'previous' link element.
  • nextText - (string, defaults to '&raquo;') A string indicating the HTML for the 'next' link element.
